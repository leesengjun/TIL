### 깃
깃이란 간단하게 말하면 코드 변경을 기록하고 관리하는 시스템이다
### 깃이 필요한 이유
코드 작업하다보면 수정하다가 망하거나 예전 버전으로 돌아가고 싶고, 여러 명이 같이 작업할수 없는 문제가 있다 GIt은 그 문제를 해결해준다.
### 핵심기능
코드 변경 이력 저장 (언제든지 돌아갈 수 있다)
백업 코드 날아가도 복구가 가능하다 
협업 여러 명이 같이 개발 가능하다 
### 기본 용어 
commit--> 변경사항 저장 (세이브)
push--> 온라인에 업로드 
pull ---> 최신 코드 가져오기 
### GItHub란 
GitHub는 Git을 온라인에서 사용하는 서비스이다. 쉽게 말하면 GitHub는 개발자들이 코드를 온라인에 저장하고, 변경 이력을 관리하며, 여러 사람과 함께 협업할수 있도록 도와주는 웹 기반 서비스이다
### 간단 흐름
git init        # 시작
git add .       # 파일 추가
git commit -m "저장"   # 저장
git push        # 업로드
### GItHub의 중요성
개발자의 필수스킬이다
취업에서도 거의 필수이
