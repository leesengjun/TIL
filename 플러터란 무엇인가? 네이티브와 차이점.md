### 플러터란 무엇인가?
플러터는 구글에서 만든 앱 개발 프레임워크이다. 하나의 코드로 여러 플랫폼 앱을 만드는 도구이다.
### 플러터로 무엇을 만들수 있을까?
플러터로 만들 수 있는 것들 안드로이드 앱, IOS ,앱 웹사이트 ,데스크톱 앱
### 사용하는 언어 
다트
### 특징
UI 직접 그림 (빠름)
핫 리로드 (코드를 바꾸면 바로 반영된다)
디자인 자유도가 높다.
###네이티브 앱이란
플랫폼 전용으로 만드는 방식 예: Android-> Kotlin / Java ios-> Swift
### Flutter 와 네이트브 차이
개발언어 플러터는 다트라는 언어를 배우고 네이티브는 코틀린과 스위프트를 배운다 
코드는 플러터는 하나로 여러 플랫폼을 만들수있고 네이티브는 플랫폼별 따로 코드를 쓴다
개발속도는 플러터가 네이티브의 비해 빠르다
ui --> 플러터는 직접 그리고 네이트브는 Os 기본 UI 에서 한다 
| 항목    | Flutter    | 네이티브           |
| ----- | ---------- | -------------- |
| 개발 언어 | Dart       | Kotlin / Swift |
| 코드    | 하나로 여러 플랫폼 | 플랫폼별 따로        |
| 성능    | 좋음         | 가장 좋음          |
| 개발 속도 | 빠름         | 느림             |
| UI    | 직접 그림      | OS 기본 UI       |
